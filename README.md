# NodeLogic

    MouseButtonLeft - move nodes,connect and select

    MouseButtonRight - change zoom

    MouseButtonMiddle - move camera

    D-duplicate node

    Delete-delete node

    And
    Or
    Not
    Nand
    Nor
    Xor
    Xnor
    JK trigger
    SR trigger
    Push button
    Toggle button
    Delay
    Pulse
    Clock
    Bit display
    RGB display with 9 colors
    Label
